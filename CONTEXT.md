# Project Context

## Terms

- **Wallpaper runtime**: the Module that owns wallpaper selection integration, apply helpers, DMS/greeter sync, and Matugen downstream hooks.
- **Check registry**: the Module that owns validation check names, host/support groups, and CI/docs alignment.
- **USB installer workflow**: the Module that owns persistent USB formatting, `update-usb`, squashfs staging, install verification, and cleanup.
